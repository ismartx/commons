## commons
commons is a common utils project.

### usage
```
<mirror>
    <id>nexus</id>
    <mirrorOf>*</mirrorOf>
    <name>Private Nexus</name>
    <url>http://repo.dingdingyisheng.mobi/repository/maven-public</url>
</mirror>
```

```
<dependency>
     <groupId>org.smartx</groupId>
     <artifactId>commons</artifactId>
     <version>1.0</version>
</dependency>
```
